# Public-safe boundary

This repository releases a narrow claim-admissibility proof surface and withholds the broader private runtime architecture.

## Safe to publish

- Generated claims are not facts.
- Completion claims require execution evidence.
- Approval is not execution.
- Evidence makes a claim admissible.
- Minimal browser and Python proof.

## Not included in this repository

- Private runtime internals
- Non-public personal materials
- Private source material
- Local filesystem evidence dumps
- Company-specific accusations
- Non-public source materials
- Confidential implementation details
